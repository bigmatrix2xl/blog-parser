# Старт через Cursor

Открой в Cursor всю папку проекта целиком.

Используй только эти файлы:
- `START_HERE_CURSOR.md`
- `CURSOR_PILOT_PROMPT.txt`
- `CURSOR_AGENT_RULES.md`
- `CURSOR_RUNBOOK.md`
- `kuvalda_blog_audit_scraper.py`
- `requirements.txt` (если его создаст/обновит Cursor)
- остальные файлы как справочные

Что делать:
1. Открой папку в Cursor.
2. Открой чат/agent.
3. Вставь текст из `CURSOR_PILOT_PROMPT.txt`.
4. Дай Cursor выполнить пилот на 5–8 статей.
5. Дождись реального результата: папка **`kuvalda_master/`** — картинки, **`kuvalda_image_report.html`**, `kuvalda_articles.csv`.

Ничего из существующих файлов вручную переписывать не нужно.
